# L5-Mission1
To run node.js server for javascript file:

npm init -y <br>
npm i express<br>
npm nodemon -D<br>
set up a script in package.json:<br>
"scripts": {<br>
    "start": "nodemon turnersAPIs.js",<br>
    "test": "echo \"Error: no test specified\" && exit 1"<br>
  },<br>
  start server: npm start. (this will start nodemon)<br>
