# L5-Mission1
To run node.js server for javascript file:

npm init -y
npm i express
npm nodemon -D
set up a script in package.json:
"scripts": {
    "start": "nodemon turnersAPIs.js",
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  start server: npm start. (this will start nodemon)
